  • If you can find some letters which will not be present in main result. Select them in ELIMINATED LETTER section. Click once to select. Click again to de-select that letter, if you have selcted that mistakenly.
  • If you can find some letters which will be present in main result, but you have set them in wrong position. You can fill those in IMPROPER POSITION section. Fill letters based on their position box. If there are multiple letters for one position, then fill them in comma seperated way. [e.g. P    A,P       ]
  • If you can find some letters which will be present in main result, but you have set them in correct position. You can fill those in PROPER POSITION section. Fill letters based on their position box. [e.g. R    C       ]
  • Click on Generate Possible Solution to generate possible word solution
  • Click Clear to clear all the filled data in ELIMINATED LETTER ,IMPROPER POSITION, PROPER POSITION
  • If you click the result outcomes , in new tab you can see it's meaning in google automatically.
